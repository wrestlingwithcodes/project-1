This is a modular, terminal-based console application to practice the fundamentals of Python.

#Features
Main menu includes a dice roller 

Features

### Main Menu
<img src="https://github.com/wrestlingwithcodes/project-1/blob/main/images_for_README/mainmenu.PNG" alt="mainmanuimage" width="209" height="145">

* **D&D Dice Roller:** A versatile roller for all standard polyhedral dice (d4-d100).
* **Secure Password Generator:** Create randomized, high-entropy passwords with custom length constraints.
* **Turtle Race:** A graphical betting game built with the `turtle` library.
<img src="https://github.com/wrestlingwithcodes/project-1/blob/main/images_for_README/turtlerace.PNG" alt="mainmanuimage" width="500" height="230">

### Converter Menu
<img src="https://github.com/wrestlingwithcodes/project-1/blob/main/images_for_README/conversions.PNG" alt="mainmanuimage" width="259" height="119">

A dedicated sub-menu for quick calculations across three domains:
* **Length:** Convert between Metric and Imperial units.
* **World Time Zone Converter:** View live, synchronized time across major global regions (NYC, London, Istanbul, Tokyo) using the `pytz` library.
* **Temperature:** Switch between Celsius and Fahrenheit.

### Games menu
<img src="https://github.com/wrestlingwithcodes/project-1/blob/main/images_for_README/games.PNG" alt="mainmanuimage" width="191" height="127">

A collection of interactive mini-games, including:
* **Word Guesser:** A classic word challenge.
<img src="https://github.com/wrestlingwithcodes/project-1/blob/main/images_for_README/wordguesser.PNG" alt="mainmanuimage" width="191" height="127">
* **Number Guesser:** A simple "Higher or Lower" game.
* **Tic-Tac-Toe:** A 2-player strategy game played directly in the console.
